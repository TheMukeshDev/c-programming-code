PPS BCS 101 - LIST OF IMPORTANT PORGRAMS

1. WAP to find the greatest of three numbers.
2. WAP that finds whether a given number is even or odd.
3. WAP that tells whether a given year is a leap year or not.
4. WAP that accepts marks of five subjects and finds percentage and prints
grades according to the following criteria:
a. Between 90-100%-----Print ‘A’
b. 80-90%-----------------Print ‘B’
c. 60-80%-----------------Print ‘C’
d. Below 60%-------------Print ‘D’
5. WAP to find the factorial of a given number using recursion.
6. WAP to print the Fibonacci series.
7. WAP to check whether the entered number is prime or not.
8. WAP to find the reverse of a number.
9. WAP to print Armstrong numbers from 1 to 100.
10. WAP to add and multiply two matrices of order n*n.
11. WAP to swap two numbers using call by value and call by reference
12. WAP to search an element in an array using Linear Search and Binary
Search.
13. WAP to sort the elements of the array in ascending order using Bubble
Sort, Insertion Sort and Selection Sort technique
